# SubTrack Mobile

Мобильное приложение SubTrack для iOS и Android на базе Expo и React Native.

## Возможности

- регистрация и вход
- просмотр дашборда
- управление подписками
- ближайшие списания
- уведомления
- аналитика расходов
- импорт данных
- интеграции
- настройки приложения

## Стек

- React Native
- Expo
- TypeScript
- Axios
- React Navigation
- AsyncStorage

## Требования

- Node.js 18+
- npm
- Expo Go или Android Studio / Xcode

npm install
